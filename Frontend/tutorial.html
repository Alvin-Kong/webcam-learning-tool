Tutorial:

In both the Learning Tool and Playground modes, you may use your mouse to draw on the canvas.

In order to enable webcam drawing:

- Ensure the API is running and the correct selected object is being tracked.

- Press the 'M' key to activate the webcame drawing cursor (purple square).

- Press the 'Enter' key to activate webcam drawing.

- Press the 'Spacebar' key to deactivate webcam drawing and the cursor.
